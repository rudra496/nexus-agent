# Contributing to NexusAgent

First off, thank you for considering contributing to NexusAgent! We're building the most private, efficient, and intelligent self-evolving local agent, and we'd love your help.

## Getting Started
1. Fork the repo and create your branch from `main`.
2. Run `pip install -e .` to install NexusAgent in editable mode.
3. Test your changes using a local Ollama model.

## Submitting a Pull Request
1. Ensure your code follows PEP-8 (we use `black` and `flake8`).
2. Add tests for any new logic.
3. Update `README.md` or `docs/` if your changes add new features.

Let's build the future of local AI together! 🚀
