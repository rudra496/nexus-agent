# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a core feature of NexusAgent, as it is designed to be a completely private, local-first LLM orchestrator. 

If you discover any security vulnerabilities, especially concerning unauthorized data exfiltration or sandbox escapes, please report them by emailing `security@nexusagent.local` or opening a draft security advisory on GitHub. We pledge to investigate and resolve all issues within 48 hours.
