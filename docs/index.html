Why Choose NexusAgent?

Built for the future of on-device AI. Stop sending your proprietary code to the cloud.

100% Local & Private

Powered by Ollama and LiteLLM. No API keys required. Your code never leaves your machine.

Self-Evolving Skills

If Nexus lacks a tool, it writes it dynamically in Python, saving it to your Skill Tree for future reuse.

GraphRAG Memory

Nexus builds a local NetworkX graph of your codebase, providing perfect semantic context without token bloat.

The Skill Tree Ecosystem

Nexus is born with a minimal set of tools. When you ask it to perform a complex task, like parsing proprietary logs or analyzing specific database schemas, it uses the local LLM to write a custom Python script.

It then saves this script permanently into your .nexus/skills/ directory. The next time you ask a similar question, Nexus instantly retrieves and executes the custom skill. It gets smarter the more you use it.

Get Started in Seconds

# 1. Install the package via pip
pip install nexus-agent

# 2. Start your local Ollama instance (we recommend llama3)
ollama run llama3

# 3. Initialize the GraphRAG memory in your project
nexus evolve

# 4. Give Nexus a task
nexus run "Find bugs in my src/ folder and fix them"
