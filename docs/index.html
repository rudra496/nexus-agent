The Self-Evolving Local AI Agent

Zero configuration. Absolute privacy. Nexus runs entirely on your machine, writes its own skills, and evolves with your codebase.

Star on GitHub Quick Start

100% Local & Private

Powered by Ollama and LiteLLM. No API keys required. Your code never leaves your machine.

Self-Evolving

Nexus learns from your workspace via local GraphRAG and generates its own reusable Skill Trees.

Zero-Config

Just run `nexus run` in any directory and watch it automate your terminal workflow instantly.

Get Started in Seconds

# Install the package
pip install nexus-agent

# Run a local model with Ollama
ollama run llama3

# Give Nexus a task
nexus run "Find bugs in my src/ folder and fix them"
